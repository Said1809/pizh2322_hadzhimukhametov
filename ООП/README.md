<h3>Задание:</h3>
<h4> Класс «Автомобиль» и дочерний класс «Автобус»</h4>
<p>Экземпляр класса имеет координаты своего положения и угол, описывающий направление движения. Он может быть изначально поставлен в любую точку с любым направлением (конструктор), может проехать в выбранном направлении определённое расстояние и может повернуть, то есть изменить текущее направление на любое другое<p>
<p>- Реализуйте класс автомобиля, а также класс, который будет описывать автобус. Кроме того, что имеется у автомобиля, у автобуса должны быть поля, содержащие число пассажиров и количество полученных денег, изначально равные нулю.<p> 
<p>-  Также должны быть методы «войти» и «выйти», изменяющие число пассажиров.<p>
<p>- Наконец, метод move должен быть переопределён, чтобы увеличивать количество денег в соответствии с количеством пассажиров и пройденным расстоянием.</p>
